Get started
Version —

1. Connect Dropbox
Press ⚙ → Connect Dropbox. A Dropbox login page opens — grant access. The player connects automatically and fetches your tracks. The token renews in the background; you never need to do this again.

2. Folder structure
Place tracks in subfolders — each folder becomes a pack/playlist. Add an image named cover.jpg to the folder for cover art.

3. Offline
Open a pack and press "Download pack". Files are saved locally in the browser and persist across sessions. A green checkmark shows which tracks are saved.

4. A/B comparison mode
Press the A/B button in the playlist to activate comparison mode. Each track gets A and B buttons — press to assign tracks to each slot. In the expanded player, A and B are shown with waveforms; press a section to switch which track plays. The track loops automatically so you can listen at leisure. Press A/B again to exit the mode.
